    https://www.random.org/integers/

2. 입력 박스에 3, 1, 연습문제 마지막 번호, 5를 입력하고, Get Numbers 버튼을 누른다. 

3. 세 개의 숫자가 나오면, 그 숫자에 해당하는 3장 연습문제를 풀어서 제출한다.
